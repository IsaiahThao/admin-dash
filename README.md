# Admin Dashboard

A front-end admin dashboard built with HTML and CSS to demonstrate strong understanding of layout systems, component structure, and modern UI styling principles.

**Live Demo:** [https://isaiahthao.github.io/admin-dash/](https://isaiahthao.github.io/admin-dash/)

---

## Project Purpose

This project was created as a portfolio piece to showcase proficiency in:

* Semantic HTML structure
* CSS Grid for large-scale page layout
* Flexbox for component-level alignment
* Responsive design principles
* Reusable component architecture
* Clean spacing and visual hierarchy

The focus of this project is layout precision, maintainable styling, and thoughtful UI composition â€” not backend functionality.

## Features

* Multi-column dashboard layout using CSS Grid
* Responsive project card grid
* Reusable card component pattern
* Sidebar with structured announcement and trending sections
* Header layout with search and user controls
* Consistent spacing, shadows, and typographic hierarchy
* Organized, scalable CSS structure

This project reflects practical front-end development skills and attention to clean, maintainable UI implementation.
